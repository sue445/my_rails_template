# Rails3 Application Template

## Usage
``` sh
rails new your_project_name -m https://raw.github.com/sue445/my_rails_template/master/template.rb
```

## Info
### Default install
* rails
* rspec
* pry
* spork
* guard

### Optional install
* Jenkins CI tools
 * simplecov
 * rails_best_practices
 * some build scripts
* capistrano
* twitter-bootstrap-rails
